Getting Started

This is a simplified landing page because the full application requires Node.js to be installed.

To run the full application:

  1. Download and install Node.js (LTS version recommended)
  2. During installation, make sure to check the option to add Node.js to your PATH
  3. Close and reopen your command prompt
  4. Run the start.bat file in this directory
  5. Select option 1 to install dependencies
  6. Then select option 2 to start the application

Alternative Method

If you can't install Node.js, you can use a local server extension:

  1. Install the Web Server for Chrome extension
  2. Open the extension and select this folder
  3. Click the URL shown (usually http://127.0.0.1:8887)
